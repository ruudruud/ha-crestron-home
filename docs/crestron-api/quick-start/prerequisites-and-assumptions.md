# Prerequisites and Assumptions

This document assumes the following knowledge prerequisites prior to working with the Crestron Home OS REST API:

- The programmer has a working knowledge of API programming conventions, including how to format HTTPS requests to transfer data to and from the device.
- The programmer has a working knowledge of Crestron Home system settings and protocols.
- The Crestron Home controller has been installed and is running on the network. For more information, refer to the [Crestron Home OS Product Manual](https://www.crestron.com/docs/8525).
